# 💳 Digital Wallet System (Spring Boot + AI)

## 📌 Overview

A scalable Digital Wallet Web Application built using Java and Spring
Boot.\
Users can add money, send money, and track transactions, with optional
AI-powered insights and search.

------------------------------------------------------------------------

## 🚀 Features

### Core Features

-   Wallet management
-   Add money (mock)
-   Send money
-   Transaction history
-   Balance validation

### AI Features

-   Smart insights
-   Natural language queries
-   Future: RAG-based search

------------------------------------------------------------------------
